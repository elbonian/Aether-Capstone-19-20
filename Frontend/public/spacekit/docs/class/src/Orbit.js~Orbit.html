    Orbit

    A class that builds a visual representation of a Kepler orbit.

    Example:

    const orbit = new Spacekit.Orbit({
  ephem: new Spacekit.Ephem({...}),
  options: {
    color: 0xFFFFFF,
    eclipticLineColor: 0xCCCCCC,
  },
});
